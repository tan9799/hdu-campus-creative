# 短命令系统说明 v5

## 可用命令列表

| 命令 | 别名 | 功能 | 示例 |
|------|------|------|------|
| `/random` | `/rand` | 完全随机选择风格（忽略用户偏好，但仍遵守排除列表） | `/random` |
| `/retry` | `/again` | 重新生成上一次的输入和风格，不改变已收集风格 | `/retry` |
| `/retry <index>` | - | 多图模式下重试指定索引（如 `/retry 2`） | `/retry 2,3` |
| `/stats` | `/info` | 显示集卡进度、已收集风格、挑战次数、总生成数 | `/stats` |
| `/save` | - | 生成恢复码（8位字母数字），保存当前进度至内存（24h有效） | `/save` → `恢复码: A3fG8kL2` |
| `/load <码>` | `/restore` | 恢复之前保存的进度 | `/load A3fG8kL2` |
| `/reset` | `/clear` | 重置当前会话所有进度（收集列表、偏好、挑战计数） | `/reset` |
| `/help` | `/?` | 显示本帮助信息 | `/help` |
| `/gallery` | - | 查看历史作品（最多20张，支持分页） | `/gallery page=2` |
| `/hot` | `/popular` | 查看热门作品（近7天点赞前10） | `/hot` |
| `/inspire` | `/idea` | 浏览灵感集市（随机20个公开作品） | `/inspire` |
| `/inspire <id>` | - | 查看某个灵感作品的详细信息并一键复刻 | `/inspire hdu_abc123` |
| `/theme <节日>` | - | 切换主题（校庆、新年、毕业季），影响UI和文案 | `/theme 校庆` |
| `/duel @对手` | - | 发起风格对战，两人各生成一张，AI 模拟投票决胜 | `/duel @zhangsan` |
| `/timecapsule` | `/future` | 生成未来明信片（需输入日期） | `/timecapsule 2028-06-30` |
| `/privacy` | - | 切换作品默认公开/私密 | `/privacy private` |
| `/daily` | - | 获取今日推荐场景（基于用户未收集风格） | `/daily` |
| `/achievements` | - | 查看已获得的成就列表 | `/achievements` |
| `/share` | - | 模拟分享当前作品（返回分享链接） | `/share` |

## 实现细节
- 命令匹配忽略大小写和首尾空格。
- 恢复码存储：内存 `Map<string, object>`，有效期 24 小时，自动清理。
- 恢复成功后提示“进度已恢复”，并显示当前集卡列表和进度条。
- `/duel` 需要对方在线且同意（自动发送挑战请求，对方回复 `/accept` 后开始）。
- `/timecapsule` 日期范围：当前时间 ~ 当前时间+10年，超出则提示“请选择合理未来日期”。
