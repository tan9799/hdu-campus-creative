# 口语清洗词汇库与规则 v2

## 语气词（需移除）
**中文**：嗯、啊、哦、呃、那个、这个、然后、就是、其实、额、嘛、啦、哈、哎、哎呀、算了、就是说  
**英文**：well, um, uh, like, actually, basically, so, you know

## 重复标点正则
- `[!?！？]{2,}` → `!` 或 `？`（保留一个）
- `[.]{2,}` → `.`
- `[ ]{2,}` → ` `（单个空格）

## 噪音词（可选移除）
- 礼貌前缀：“我想”、“帮我”、“请”、“能不能”、“可不可以” → 保留或移除（不影响主体提取）
- 无意义填充：“那个那个”、“然后然后” → 移除

## 示例
输入：  
“嗯...那个...图书馆5楼靠窗，夕阳，复习的第100天！！！”

清洗后：  
“图书馆5楼靠窗，夕阳，复习的第100天！”

## 偏好提取正则（多语言）
- 不喜欢风格：`不喜欢(.*?风)` 或 `不要(.*?风)` 或 `hate (.*?style)` 或 `don't like (.*?)`
- 喜欢风格：`喜欢(.*?风)` 或 `优先(.*?风)` 或 `love (.*?)` 或 `prefer (.*?)`
- 重置偏好：`重置偏好` 或 `清除偏好` 或 `reset preferences`

## 多图数量提取
- 匹配 `(\d+)[张幅个]` 或 `(\d+) images`
- 匹配汉字数字：`两` → 2, `三` → 3, `四` → 4, `五` → 5, `几` → 3（默认）
- 注意：若用户同时输入“两张”和“三个风格”，取最大数，并提示“将生成3张图片”。

## 实体链接增强（杭电专属）
- 地标别名：月雅湖 → Yanya Lake, 问鼎广场 → Wending Square
- 活动缩写：ACM → ACM程序设计竞赛，电设 → 电子设计竞赛
- 自动补全：若用户只写“图书馆”，默认为“杭电图书馆（12层主楼）”
